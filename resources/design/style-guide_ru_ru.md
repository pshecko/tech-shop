# Botiga Tech — Графическая карта

> Документ дизайна · v1.0 · Март 2026  
>  
> Этот документ определяет визуальную систему Botiga Tech. В нем содержатся все дизайнерские решения, которые должен реализовать фронтенд-разработчик: цветовая палитра, типография, отступы, компоненты и интерактивные поведения. Любые значения, не указанные здесь, необходимо согласовывать перед их созданием.

## 1. Макет и сетка

### Контейнер

Содержимое всех страниц горизонтально центрировано внутри контейнера с максимальной шириной **1200px** и боковыми отступами **24px** на мобильных устройствах и **40px** на десктопе. На экранах шире контейнера видно фон страницы по обеим сторонам.

### Точки перелома (Breakpoints)

| Название   | Условие             | Основное поведение              |
| ---------- | ------------------- | ------------------------------ |
| **Мобильный** | base (без query)  | 1 колонка. Минимальная ширина: 375px |
| **Планшет**   | `min-width: 768px` | 2 колонки в каталоге           |
| **Десктоп**   | `min-width: 1024px`| 3 колонки в каталоге           |

> **Mobile-first.** Базовые стили пишутся для мобильных устройств. Медиа-запросы только расширяют или корректируют их для больших экранов. Никогда не используйте `max-width`.

### Каталог — сетка продуктов

- Мобильный: 1 колонка, расстояние между карточками **16px**  
- Планшет: 2 колонки, расстояние **20px**  
- Десктоп: 3 колонки, расстояние **24px**

## 2. Цветовая палитра

### Основные цвета

| Токен             | HSL значение          | Образец                     | Применение                        |
| ----------------- | --------------------- | --------------------------- | -------------------------------- |
| `--color-bg`       | `hsl(220, 14%, 96%)`  | Очень светло-серый, почти белый | Фон страницы                     |
| `--color-surface`  | `hsl(0, 0%, 100%)`    | Чисто белый                 | Карточки, модальные окна, формы  |

### Фирменные цвета

| Токен                 | HSL значение          | Применение                                                       |
| --------------------- | --------------------- | ---------------------------------------------------------------- |
| `--color-primary`      | `hsl(221, 83%, 53%)`  | Средний синий — кнопки CTA, активные ссылки, граница фокуса      |
| `--color-primary-dark` | `hsl(221, 83%, 40%)`  | Темно-синий — состояние `:hover` основной кнопки                 |
| `--color-primary-bg`   | `hsl(221, 83%, 97%)`  | Очень бледный синий — фоны акцентов или мягких баннеров          |

### Цвета текста

| Токен                   | HSL значение          | Контраст на surface | Применение                                                |
| ----------------------- | --------------------- | ------------------- | --------------------------------------------------------- |
| `--color-text-primary`  | `hsl(222, 47%, 11%)`  | AAA                 | Заголовки, названия продуктов, цены                       |
| `--color-text-secondary`| `hsl(220, 9%, 46%)`   | AA                  | Описания, вспомогательный текст, заголовки таблиц        |
| `--color-text-muted`    | `hsl(220, 9%, 65%)`   | Только крупный текст | Плейсхолдеры, деактивированные метки                      |

> Цвет muted **не соответствует стандарту AA** на белом фоне при небольших размерах. Используйте только при `font-size` ≥ 18px или с жирным начертанием.

### Цвета границ

| Токен                  | HSL значение          | Применение                                                    |
| ---------------------- | --------------------- | -------------------------------------------------------------- |
| `--color-border`       | `hsl(220, 13%, 91%)`  | Границы карточек, разделители таблиц, неактивные поля ввода    |
| `--color-border-focus` | `hsl(221, 83%, 53%)`  | Граница поля при фокусе (тот же цвет, что и `--color-primary`) |

### Значки наличия на складе

Каждый значок имеет бледный фон (10% непрозрачности основного цвета) и текст насыщенного тона. Индикация:

| Токен                 | HSL значение          | Статус                         | Условие           |
| --------------------- | --------------------- | ------------------------------ | ----------------- |
| `--color-badge-in`    | `hsl(142, 71%, 45%)`  | Зеленый — **В наличии**          | от 3 и более штук |
| `--color-badge-low`   | `hsl(25, 95%, 53%)`   | Оранжевый — **Мало штук**        | 1 или 2 штуки     |
| `--color-badge-out`   | `hsl(0, 84%, 60%)`    | Красный — **Нет в наличии**      | 0 штук            |

Фон значка того же тона с `opacity: 0.12` или с использованием альфа-канала в `hsl` `/ 12%`.

## 3. Типография

### Шрифтовая семья

**Inter** — без засечек, высокая читаемость, оптимизирован для экранов.

- Шрифт: [Google Fonts — Inter](https://fonts.google.com/specimen/Inter)  
- Используемые веса: **400** (Regular) · **500** (Medium) · **600** (Semibold) · **700** (Bold)  
- Запасной шрифт: `system-ui, -apple-system, sans-serif`

В интерфейсе не используется никаких других шрифтов. Не применять засечки, моноширинные или системные шрифты по умолчанию, такие как Arial или Comic Sans.

### Типографическая шкала

Базовая система — **1rem = 16px** (корневой размер шрифта в браузере, не менять в `html`).

| Шаг    | rem       | px   | Обычно                          | Основное применение                                |
| ------ | --------- | ---- | ------------------------------- | ------------------------------------------------- |
| `xs`   | 0.75 rem  | 12px | Semibold 600                   | Значки, подзаголовки, метки наличия                |
| `sm`   | 0.875 rem | 14px | Regular 400                   | Вспомогательный текст, заголовки таблиц, плейсхолдеры |
| `base` | 1 rem     | 16px | Regular 400                   | Абзацы, описания, элементы списков                 |
| `lg`   | 1.125 rem | 18px | Semibold 600                 | Название продукта в карточке, цена                  |
| `xl`   | 1.5 rem   | 24px | Bold 700                     | Заголовок раздела — `<h2>`                          |
| `2xl`  | 2 rem     | 32px | Bold 700                     | Заголовок страницы — `<h1>`                         |

### Межстрочный интервал

| Контекст          | Значение | Применение                                     |
| ----------------- | -------- | ---------------------------------------------- |
| Заголовки (`h1`, `h2`) | `1.2`    | `line-height: 1.2`                             |
| Текст             | `1.6`    | `line-height: 1.6` — значение по умолчанию для `body` |

## 4. Отступы

Система кратна 8: все внутренние (padding) и внешние (margin, gap) отступы — кратны **4px**. Нечётные значения (например, 6px) запрещены.

| Токен         | rem       | px   | Типичное применение                                  |
| ------------- | --------- | ---- | --------------------------------------------------- |
| `--space-1`   | 0.25 rem  | 4px  | Минимальный отступ между inline элементами          |
| `--space-2`   | 0.5 rem   | 8px  | Padding значков · расстояние между иконкой и текстом |
| `--space-3`   | 0.75 rem  | 12px | Внутренний padding полей ввода                        |
| `--space-4`   | 1 rem     | 16px | Боковой padding карточки на мобильных · минимальный gap в сетке |
| `--space-6`   | 1.5 rem   | 24px | Внутренний padding карточки на десктопе · gap в сетке |
| `--space-8`   | 2 rem     | 32px | Отступы между секциями на мобильных                   |
| `--space-12`  | 3 rem     | 48px | Отступы между секциями на десктопах                    |
| `--space-16`  | 4 rem     | 64px | Вертикальные отступы блоков каталога                   |

## 5. Границы и тени

### Радиус границ

| Токен           | Значение | Применение                     |
| --------------- | -------- | ------------------------------ |
| `--radius-sm`   | 6px      | Кнопки, поля ввода             |
| `--radius-md`   | 12px     | Карточки продуктов             |
| `--radius-lg`   | 16px     | Акцентные карточки или модальные окна |
| `--radius-full` | 9999px   | Значки, таблетки, круглые аватары  |

### Тени

Тени нейтрального цвета (тот же тон, что и `--color-text-secondary`) с пониженной прозрачностью. Не использовать насыщенные цветные тени.

| Токен          | Значение                           | Применение                              |
| -------------- | --------------------------------- | -------------------------------------- |
| `--shadow-sm`  | `0 1px 2px hsl(220 9% 46% / 8%)`  | Карточка в обычном состоянии          |
| `--shadow-md`  | `0 4px 12px hsl(220 9% 46% / 12%)`| Карточка при `:hover` · приподнятые элементы |
| `--shadow-lg`  | `0 8px 24px hsl(220 9% 46% / 16%)`| Модальные окна, выпадающие меню        |

## 6. Компоненты

### 6.1 Карточка продукта

```
┌─────────────────────────────┐
│  ┌───────────────────────┐  │
│  │                       │  │
│  │    изображение товара │  │  ← aspect-ratio 4/3 · object-fit: cover
│  │                       │  │
│  └───────────────────────┘  │
│                             │
│  Название продукта          │  ← размер шрифта lg · semibold · цвет primary
│  Краткое описание (опц.)   │  ← размер шрифта sm · regular · цвет secondary
│                             │
│  169 €                  [●] │  ← цена: lg · bold · цвет primary
│                             │     значок наличия: выровнен по правому краю
│  ┌──────────────────────┐   │
│  │   Добавить в корзину │   │  ← основная кнопка · полная ширина
│  └──────────────────────┘   │
└─────────────────────────────┘
```

**Спецификации:**

| Свойство           | Значение                                |
| ------------------ | -------------------------------------- |
| Фон                | `--color-surface`                       |
| Граница            | `1px solid --color-border`              |
| Радиус             | `--radius-md`                           |
| Тень в покое       | `--shadow-sm`                           |
| Тень при `:hover`  | `--shadow-md`                           |
| Поднятие при `:hover` | `transform: translateY(-2px)`         |
| Переходы           | `box-shadow 0.2s ease, transform 0.2s ease` |
| Внутренний отступ  | `--space-4` (мобильный) · `--space-6` (десктоп) |

Изображение занимает всю ширину карточки. На мобильных устройствах карточка не имеет бокового скругления (идет от края до края). На десктопе есть.

### 6.2 Значок наличия на складе

Встроенный элемент, показывающий доступность товара. Всегда видим в карточке.

```
 ● В наличии     ← зеленый
 ● Мало штук     ← оранжевый
 ● Нет в наличии ← красный
```

| Свойство      | Значение                                               |
| ------------- | ----------------------------------------------------- |
| Display       | `inline-flex` с `align-items: center` и `gap: --space-1` |
| Размер шрифта | `--font-size-xs`                                      |
| Толщина шрифта| `--font-weight-semibold`                              |
| Padding       | `--space-1` вертикально · `--space-2` горизонтально  |
| Радиус границ | `--radius-full`                                       |
| Фон           | цвет значка с 12% прозрачности                        |
| Текст         | цвет значка без прозрачности                          |

### 6.3 Основная кнопка

Главное действие карточки: «Добавить в корзину».

| Состояние        | Фон                    | Текст               | Граница                                                  |
| ---------------- | ---------------------- | ------------------- | --------------------------------------------------------- |
| Обычное          | `--color-primary`      | белый               | отсутствует                                              |
| `:hover`         | `--color-primary-dark` | белый               | отсутствует                                              |
| `:focus-visible` | `--color-primary`      | белый               | `outline: 2px solid --color-primary` с отступом 2px      |
| `:disabled`      | `--color-border`       | `--color-text-muted` | отсутствует                                              |

| Свойство       | Значение                                  |
| -------------- | ---------------------------------------- |
| Padding        | `--space-3` вертикально · `--space-6` горизонтально |
| Радиус границ  | `--radius-sm`                            |
| Размер шрифта  | `--font-size-base`                       |
| Толщина шрифта | `--font-weight-semibold`                 |
| Переходы       | `background-color 0.15s ease`            |
| Курсор         | `pointer`                                |

### 6.4 Таблица корзины

Таблица занимает всю ширину контейнера. Без внешних рамок — только горизонтальные разделители между строками.

```
┌──────────────────┬──────┬────────────┬──────────┐
│ Товар            │ Кол-во│ Цена/шт.   │  Итого   │  ← <thead>: фон --color-bg
├──────────────────┼──────┼────────────┼──────────┤
│ Видеокарта       │  1   │  169,00 €  │ 169,00 € │  ← <tbody>
├──────────────────┼──────┼────────────┼──────────┤
│ Оперативная память│  2   │   21,00 €  │  42,00 € │
├──────────────────┴──────┴────────────┼──────────┤
│ Итого                               │ 211,00 € │  ← <tfoot>: жирный шрифт
└─────────────────────────────────────┴──────────┘
```

| Зона           | Фон               | Текст                                      | Нижняя граница  |
| -------------- | ----------------- | ------------------------------------------ | --------------- |
| `<thead>`      | `--color-bg`      | `--color-text-secondary` · sm · semibold   | `--color-border`|
| Строки `<tbody>`| `--color-surface` | `--color-text-primary` · base · regular    | `--color-border`|
| `<tfoot>`      | `--color-surface` | `--color-text-primary` · base · **bold**   | отсутствует     |

Внутренний отступ ячеек: `--space-3` вертикально · `--space-4` горизонтально.

### 6.5 Заголовок сайта

Хедер фиксированный (`position: sticky; top: 0`) с фоном `--color-surface` и тонкой нижней тенью (`--shadow-sm`) для обозначения прокрутки контента под ним.

```
┌──────────────────────────────────────────────────────────┐
│  Botiga Tech          Главная  Продукты  Корзина (2)      │
└──────────────────────────────────────────────────────────┘
```

- Логотип/название: `--font-size-xl` · жирный · `--color-text-primary`  
- Навигационные ссылки: `--font-size-base` · medium · `--color-text-secondary`  
- Активная/hover ссылка: `--color-primary`  
- Высота хедера: **64px**  
- Боковые отступы: как в контейнере

## 7. Взаимодействия и анимации

### Принципы

- Переходы всегда **тонкие и быстрые**: от 100 до 250 мс.  
- Кривая ускорения: `ease` для стандартных появлений/исчезновений.  
- Не использовать анимации дольше 300 мс на часто используемых элементах UI.

### Сводка интерактивных состояний

| Элемент           | `:hover`                            | `:focus-visible`                                        |
| ----------------- | ---------------------------------- | ------------------------------------------------------- |
| Карточка продукта | `translateY(-2px)` + `--shadow-md`| `outline: 2px solid --color-primary` с отступом 2px     |
| Основная кнопка   | Фон `--color-primary-dark`          | `outline: 2px solid --color-primary` с отступом 2px     |
| Навигационная ссылка | Цвет `--color-primary`             | `outline: 2px solid --color-primary` с отступом 2px     |
| Поле ввода формы  | Граница `--color-border-focus`      | Граница `--color-border-focus` + заметный `outline`    |

> Никогда не используйте `outline: none` без замены видимым индикатором. Индикатор фокуса обязателен для доступности с клавиатуры.
