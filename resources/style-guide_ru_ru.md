# Botiga Tech — Видеокарта

> Дизайн-документ · v1.0 · Март 2026
>
> Этот документ определяет визуальную систему Botiga Tech. В нем содержатся все дизайнерские решения, которые должен реализовать фронтенд-разработчик: цветовая палитра, типография, отступы, компоненты и интерактивное поведение. Любые значения, не определённые здесь, должны быть согласованы перед их созданием.

---

## 1. Макет и сетка

### Контейнер

Содержимое всех страниц центрировано по горизонтали внутри контейнера с максимальной шириной **1200px** и боковыми отступами **24px** на мобильных устройствах и **40px** на десктопе. На экранах шире контейнера фон страницы виден по обе стороны.

### Точки перелома (Breakpoints)

| Название  | Условие             | Основное поведение              |
| --------- | ------------------- | ------------------------------ |
| **Мобильный** | base (без query)   | 1 колонка. Минимальная ширина: 375px |
| **Планшет**  | `min-width: 768px`  | 2 колонки в каталоге            |
| **Десктоп** | `min-width: 1024px` | 3 колонки в каталоге            |

> **Mobile-first.** Базовые стили пишутся для мобильных. Медиа-запросы расширяют или корректируют их для больших экранов. Никогда не используйте `max-width`.

### Каталог — сетка продуктов

- Мобильный: 1 колонка, отступы между карточками **16px**
- Планшет: 2 колонки, отступы **20px**
- Десктоп: 3 колонки, отступы **24px**

## 2. Цветовая палитра

### Основные цвета

| Токен                     | Значение HSL             | Образец | Использование |
| ------------------------- | ------------------------ | ------- | ------------ |
| `--color-bg`              | `hsl(220, 14%, 96%)`     | Очень светло-серый, почти белый | Фон страницы |
| `--color-surface`         | `hsl(0, 0%, 100%)`       | Чисто белый | Карточки, модальные окна, формы |

### Фирменные цвета

| Токен                     | Значение HSL             | Использование |
| ------------------------- | ------------------------ | ------------ |
| `--color-primary`         | `hsl(221, 83%, 53%)`     | Средний синий — кнопки CTA, активные ссылки, фокусная рамка |
| `--color-primary-dark`    | `hsl(221, 83%, 40%)`     | Тёмный синий — состояние `:hover` для основной кнопки |
| `--color-primary-bg`      | `hsl(221, 83%, 97%)`     | Очень бледный синий — фоны выделений или мягких баннеров |

### Цвета текста

| Токен                      | Значение HSL             | Контраст по отношению к surface | Использование |
| -------------------------- | ------------------------ | ------------------------------ | ------------ |
| `--color-text-primary`     | `hsl(222, 47%, 11%)`     | AAA                            | Заголовки, названия продуктов, цена |
| `--color-text-secondary`   | `hsl(220, 9%, 46%)`      | AA                             | Описания, вспомогательный текст, заголовки таблиц |
| `--color-text-muted`       | `hsl(220, 9%, 65%)`      | Только для крупного текста     | Плейсхолдеры, отключённые метки |

> Текст в muted цвете **не соответствует требованиям AA** на белом фоне при малом размере шрифта. Используйте только для `font-size` ≥ 18px или с жирным начертанием.

### Цвета границ

| Токен                   | Значение HSL             | Использование |
| ----------------------- | ------------------------ | ------------ |
| `--color-border`        | `hsl(220, 13%, 91%)`     | Границы карточек, разделители таблиц, неактивные инпуты |
| `--color-border-focus`  | `hsl(221, 83%, 53%)`     | Граница инпута при фокусе (тот же цвет, что и `--color-primary`) |

### Значки наличия на складе (Badges de stock)

Каждый значок имеет бледный фоновый цвет (10% прозрачности основного цвета) и текст полного тона. Пояснения:

| Токен                   | Значение HSL             | Статус       | Условие          |
| ----------------------- | ------------------------ | ------------ | ---------------- |
| `--color-badge-in`      | `hsl(142, 71%, 45%)`     | Зелёный — **В наличии**    | 3 и более единиц     |
| `--color-badge-low`     | `hsl(25, 95%, 53%)`      | Оранжевый — **Мало единиц** | 1 или 2 единицы     |
| `--color-badge-out`     | `hsl(0, 84%, 60%)`       | Красный — **Нет в наличии** | 0 единиц           |

Фон каждого значка — тот же оттенок с `opacity: 0.12` или с использованием `hsl` с альфа-каналом `/ 12%`.

## 3. Типография

### Семейство шрифтов

**Inter** — без засечек, высокоразборчивый шрифт, оптимизированный для экранов.

- Шрифт: [Google Fonts — Inter](https://fonts.google.com/specimen/Inter)
- Используемые начертания: **400** (Regular) · **500** (Medium) · **600** (Semibold) · **700** (Bold)
- Запасные шрифты: `system-ui, -apple-system, sans-serif`

Никакие другие шрифты в интерфейсе не используются. Не использовать serif, monospace или стандартные браузерные шрифты как Arial или Comic Sans.

### Типографическая шкала

Базовый размер системы — **1rem = 16px** (корневой размер шрифта браузера, не изменять в `html`).

| Размер  | rem       | px   | Обычный вес шрифта | Основное использование                  |
| ------- | --------- | ---- | ------------------ | ------------------------------------- |
| `xs`    | 0.75 rem  | 12px | Semibold 600       | Значки, подписи, метки наличия        |
| `sm`    | 0.875 rem | 14px | Regular 400        | Вспомогательный текст, заголовки таблиц, плейсхолдеры |
| `base`  | 1 rem     | 16px | Regular 400        | Абзацы, описания, элементы списков    |
| `lg`    | 1.125 rem | 18px | Semibold 600       | Названия продуктов в карточках, цена |
| `xl`    | 1.5 rem   | 24px | Bold 700           | Заголовок раздела — `<h2>`             |
| `2xl`   | 2 rem     | 32px | Bold 700           | Заголовок страницы — `<h1>`            |

### Межстрочный интервал

| Контекст            | Значение | Применение               |
| ------------------- | -------- | ----------------------- |
| Заголовки (`h1`, `h2`) | `1.2`    | `line-height: 1.2`       |
| Основной текст       | `1.6`    | `line-height: 1.6` — значение по умолчанию для `body` |

## 4. Отступы

Система на основе 8 пикселей: все внутренние (padding) и внешние (margin, gap) отступы кратны **4px**. Нечётные значения (например, 6px) запрещены.

| Токен         | rem       | px   | Типичное использование                   |
| ------------- | --------- | ---- | --------------------------------------- |
| `--space-1`   | 0.25 rem  | 4px  | Минимальный отступ между inline элементами |
| `--space-2`   | 0.5 rem   | 8px  | Padding значков · отступ между иконкой и текстом |
| `--space-3`   | 0.75 rem  | 12px | Внутренний отступ в инпутах              |
| `--space-4`   | 1 rem     | 16px | Боковые паддинги карточек на мобильных · минимальный gap сетки |
| `--space-6`   | 1.5 rem   | 24px | Внутренний паддинг карточек на десктопах · gap сетки на десктопах |
| `--space-8`   | 2 rem     | 32px | Отступы между секциями на мобильных      |
| `--space-12`  | 3 rem     | 48px | Отступы между секциями на десктопах      |
| `--space-16`  | 4 rem     | 64px | Вертикальные отступы блока каталога      |

## 5. Границы и тени

### Радиус границы

| Токен           | Значение | Использование                    |
| --------------- | -------- | ------------------------------ |
| `--radius-sm`   | 6px      | Кнопки, инпуты                 |
| `--radius-md`   | 12px     | Карточки продукта              |
| `--radius-lg`   | 16px     | Выделенные карточки или модальные окна |
| `--radius-full` | 9999px   | Значки, pills, круглые аватары |

### Тени

Тени имеют нейтральный цвет (тот же оттенок, что и `--color-text-secondary`) с пониженной непрозрачностью. Не использовать насыщенные цветные тени.

| Токен          | Значение                                 | Использование                        |
| -------------- | --------------------------------------- | ---------------------------------- |
| `--shadow-sm`  | `0 1px 2px hsl(220 9% 46% / 8%)`        | Карточка в состоянии покоя          |
| `--shadow-md`  | `0 4px 12px hsl(220 9% 46% / 12%)`      | Карточка в `:hover` · приподнятые элементы |
| `--shadow-lg`  | `0 8px 24px hsl(220 9% 46% / 16%)`      | Модальные окна, выпадающие меню     |

## 6. Компоненты

### 6.1 Карточка продукта

```
┌─────────────────────────────┐
│  ┌───────────────────────┐  │
│  │                       │  │
│  │   изображение продукта  │  │  ← aspect-ratio 4/3 · object-fit: cover
│  │                       │  │
│  └───────────────────────┘  │
│                              │
│  Название продукта           │  ← font-size lg · semibold · text-primary
│  Краткое описание (опц.)    │  ← font-size sm · regular · text-secondary
│                              │
│  169 €                 [●]  │  ← цена: lg · bold · text-primary
│                              │     значок наличия: выравнен по правому краю
│  ┌──────────────────────┐   │
│  │ Добавить в корзину    │   │  ← основная кнопка · ширина 100%
│  └──────────────────────┘   │
└─────────────────────────────┘
```

**Спецификации:**

| Свойство      | Значение               |
| ------------- | --------------------- |
| Фон           | `--color-surface`     |
| Граница       | `1px solid --color-border` |
| Радиус        | `--radius-md`         |
| Тень в покое  | `--shadow-sm`         |
| Тень `:hover` | `--shadow-md`         |
| Подъем `:hover` | `transform: translateY(-2px)` |
| Переход       | `box-shadow 0.2s ease, transform 0.2s ease` |
| Внутренние отступы | `--space-4` (мобильный) · `--space-6` (десктоп) |

Изображение занимает всю ширину карточки. На мобильных карточка не имеет боковых скруглений (идёт от края до края). На десктопах — скругления есть.

---

### 6.2 Значок наличия

Инлайн элемент, указывающий доступность. Всегда виден в карточке.

```
 ● В наличии      ← зеленый
 ● Мало единиц    ← оранжевый
 ● Нет в наличии  ← красный
```

| Свойство     | Значение                                        |
| ------------ | ---------------------------------------------- |
| Отображение  | `inline-flex` с `align-items: center` и `gap: --space-1` |
| Размер шрифта | `--font-size-xs`                               |
| Начертание   | `--font-weight-semibold`                        |
| Паддинг     | `--space-1` вертикально · `--space-2` горизонтально |
| Радиус границы | `--radius-full`                               |
| Фон          | цвет значка с 12% прозрачностью                |
| Текст        | цвет значка 100%                                |

### 6.3 Основная кнопка

Главное действие карточки: «Добавить в корзину».

| Состояние       | Фон                | Текст    | Граница |
| --------------- | ------------------ | -------- | ------- |
| В покое         | `--color-primary`  | белый    | отсутствует |
| `:hover`        | `--color-primary-dark` | белый    | отсутствует |
| `:focus-visible`| `--color-primary`  | белый    | `outline: 2px solid --color-primary` с отступом 2px |
| `:disabled`     | `--color-border`   | `--color-text-muted` | отсутствует |

| Свойство      | Значение               |
| ------------- | --------------------- |
| Отступы      | `--space-3` вертикально · `--space-6` горизонтально |
| Радиус       | `--radius-sm`          |
| Размер шрифта | `--font-size-base`    |
| Начертание   | `--font-weight-semibold`|
| Переход      | `background-color 0.15s ease` |
| Курсор       | `pointer`              |

### 6.4 Таблица корзины

Таблица занимает всю ширину контейнера. Без внешних границ — только горизонтальные разделители между строками.

```
┌──────────────────┬──────┬────────────┬──────────┐
│ Продукт          │ Кол-во│ Цена за ед.│ Итог     │  ← <thead>: фон --color-bg
├──────────────────┼──────┼────────────┼──────────┤
│ Видеокарта       │  1   │  169,00 €  │ 169,00 € │  ← <tbody>
├──────────────────┼──────┼────────────┼──────────┤
│ Оперативная память│  2   │   21,00 €  │  42,00 € │
├──────────────────┴──────┴────────────┼──────────┤
│ Итого                                │ 211,00 € │  ← <tfoot>: жирный шрифт
└──────────────────────────────────────┴──────────┘
```

| Область     | Цвет фона         | Текст                        | Нижняя граница       |
| ----------- | ----------------- | ---------------------------- | -------------------- |
| `<thead>`   | `--color-bg`      | `--color-text-secondary` · sm · semibold | `--color-border`     |
| `<tbody>` строк | `--color-surface` | `--color-text-primary` · base · regular | `--color-border`     |
| `<tfoot>`   | `--color-surface` | `--color-text-primary` · base · **bold** | отсутствует          |

Отступы по ячейкам: `--space-3` вертикально · `--space-4` горизонтально.

---

### 6.5 Заголовок сайта

Заголовок фиксированный (`position: sticky; top: 0`) с фоном `--color-surface` и тонкой нижней тенью (`--shadow-sm`), показывающей, что содержимое прокручивается под ним.

```
┌──────────────────────────────────────────────────────────┐
│  Botiga Tech          Главная  Продукты  Корзина (2)      │
└──────────────────────────────────────────────────────────┘
```

- Логотип/название: `--font-size-xl` · жирный · `--color-text-primary`
- Навигационные ссылки: `--font-size-base` · medium · `--color-text-secondary`
- Активная/ховер ссылка: `--color-primary`
- Высота заголовка: **64px**
- Боковые паддинги: как у контейнера

## 7. Взаимодействия и анимации

### Принципы

- Переходы всегда **мягкие и быстрые**: от 100мс до 250мс.
- Кривая ускорения: `ease` для общих входов/выходов.
- Не использовать анимации длиннее 300мс на частых UI-элементах.

### Итог интерактивных состояний

| Элемент        | `:hover`                                   | `:focus-visible`                                      |
| ---------------|--------------------------------------------|------------------------------------------------------|
| Карточка товара | `translateY(-2px)` + `--shadow-md`        | `outline: 2px solid --color-primary` с отступом 2px  |
| Основная кнопка | фон `--color-primary-dark`                  | `outline: 2px solid --color-primary` с отступом 2px  |
| Навигационная ссылка | цвет `--color-primary`                  | `outline: 2px solid --color-primary` с отступом 2px  |
| Инпут формы    | граница `--color-border-focus`              | граница `--color-border-focus` + видимый `outline`  |

> Никогда не используйте `outline: none` без видимой замены. Индикатор фокуса обязателен для доступности при работе с клавиатурой.
