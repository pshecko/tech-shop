# Tech Shop

Modernización de una tienda online de informática con HTML5 y CSS3, aplicando maquetación semántica, diseño responsive mobile-first y buenas prácticas de accesibilidad.

## Descripción

Este proyecto consiste en rehacer desde cero una tienda online antigua, sustituyendo una estructura obsoleta por una interfaz moderna, limpia y adaptable a cualquier dispositivo.

El objetivo principal es construir una página de catálogo de productos con:

- HTML semántico
- CSS organizado en un único archivo
- Diseño responsive con enfoque mobile-first
- CSS Grid para la cuadrícula de productos
- Estados interactivos accesibles
- Tabla de carrito correctamente estructurada

## Objetivos del ejercicio

- Reemplazar una web antigua basada en `div` y posicionamiento manual
- Crear una estructura semántica con `header`, `nav`, `main`, `section`, `article` y `footer`
- Diseñar tarjetas de producto claras y legibles
- Aplicar variables CSS en `:root`
- Construir un layout adaptable a móvil, tablet y desktop
- Mantener un flujo de trabajo ordenado con commits atómicos y Conventional Commits

## Tecnologías

- HTML5
- CSS3
- CSS Grid
- Google Fonts

## Proceso de trabajo

El desarrollo seguirá esta secuencia:

1. Crear el README inicial
2. Construir el HTML semántico
3. Añadir reset, variables y tipografía base
4. Maquetar la versión móvil
5. Añadir breakpoints para tablet y desktop
6. Incorporar estados interactivos
7. Validar, revisar accesibilidad y publicar en GitHub Pages
